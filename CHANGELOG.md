# 更新日志

所有重要的项目变更都将记录在此文件中。

格式遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，版本号遵循 [语义化版本控制](https://semver.org/lang/zh-CN/)。

---

## [1.5.8] - 2026-06-12

### 新增

- **响应式布局设计**
  - 移动端：底部导航栏 + 功能抽屉（仿安卓应用抽屉理念）
  - PC端：顶部导航栏设计
  - 自适应内容展示布局

- **好友系统**
  - 将单向关注系统重构为双向好友系统
  - 好友请求发送、同意、拒绝功能
  - 好友备注名管理
  - 共同好友查看

- **即时聊天功能**
  - 基于 IM 服务的实时聊天
  - 会话列表管理
  - 消息发送与接收
  - 未读消息计数

- **公告弹窗系统**
  - 站点公告自动弹窗显示
  - 全局弹窗服务（Alert/Confirm/Prompt）

### 修复

- 修复前端 API 导入错误（`follow.js` 和 `chat.js`）
- 修复编译时 `GetUnreadCount` 函数重定义问题

### 变更

- **前端布局重构**
  - 创建 `AppLayout.vue` 统一管理响应式导航
  - 移除各视图组件中的重复导航栏
  - 更新首页、好友列表、聊天等页面适配新布局

- **数据库整合**
  - IM 服务与论坛端共用同一数据库实例
  - 统一用户账号系统

## [1.5.7] - 2026-06-12

### 新增

- **签到系统**
  - 用户每日签到功能
  - 连续签到天数统计
  - 签到历史记录查询

- **嵌套评论回复**
  - 支持无限层级评论回复
  - 递归删除评论（级联删除子回复）

- **API文档重构**
  - 将单一大文档拆分为模块化文件
  - 按功能模块组织（认证、文章、评论等）
  - 创建 `docs/api/` 目录及 `README.md` 索引

- **管理员功能增强**
  - 用户封禁/解封功能
  - 文章置顶/取消置顶
  - 举报管理系统

### 修复

- 修复 `UnlikeArticle` 函数中使用未定义变量 `article` 的问题
- 修复数据库计数更新的并发问题（使用 `gorm.Expr` 原子操作）
- 修复评论系统仅支持一级回复的限制
- 修复前端API路径缺少 `/api` 前缀的问题

### 变更

- **前端完全重写**
  - 迁移至 Vue3 + Vuetify 3 框架
  - 采用淡紫色主题配色方案
  - 统一全局样式和组件默认配置
  - 添加卡片悬浮效果和过渡动画
  - 实现响应式布局设计

- **API接口优化**
  - 统一所有接口路径前缀为 `/api`
  - 添加完整的错误处理和状态码
  - 完善认证机制（JWT Bearer Token）
