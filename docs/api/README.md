## 通用说明

### 认证头
所有需要认证的接口都需要在请求头中携带 JWT Token：
```
Authorization: Bearer <token>
```

### 响应格式
成功响应：
```json
{
  "message": "操作成功",
  "data": {...}
}
```

错误响应：
```json
{
  "error": "错误信息"
}
```

### HTTP状态码
| 状态码 | 描述 |
|--------|------|
| 200 | 请求成功 |
| 400 | 请求参数错误 |
| 401 | 未认证或Token无效 |
| 403 | 无权限访问 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |
